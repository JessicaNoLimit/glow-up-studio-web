1. Visión general del sistema

Glow Up Studio es una plataforma web para un centro de estética que combina una web corporativa pública con un sistema de reservas online y un CRM interno para la gestión del negocio.

El objetivo del sistema es digitalizar y organizar la gestión de citas, clientes, profesionales, salas y pagos, permitiendo que las clientas puedan reservar sus servicios online y que el centro pueda gestionar toda la actividad desde un panel de administración.

El sistema está dividido en dos grandes partes:

Zona pública (web + reservas online)
Zona interna (CRM / panel de administración)
2. Zona pública – Web y reservas online

La zona pública está pensada para las clientas y visitantes de la web.
Desde esta parte se puede:

Conocer el centro y sus servicios
Ver resultados y trabajos realizados
Consultar precios y duración de los servicios
Reservar uno o varios servicios online
Pagar una señal para confirmar la reserva
Recibir confirmación por email

La clienta no elige profesional. El sistema asigna automáticamente los servicios a los profesionales disponibles para optimizar la agenda del centro.

La clienta solo ve:

Servicios seleccionados
Día disponible
Hora de inicio
Duración total estimada
Precio total
Señal a pagar online
Importe restante a pagar en el centro

La reserva solo se confirma cuando la clienta realiza el pago de la señal.
Si no hay pago, no se registra la cita ni se bloquea el hueco en la agenda.

3. Zona interna – CRM / Panel de administración

La zona interna es utilizada por el centro para gestionar toda la actividad del negocio.

Desde el panel de administración se podrá:

Ver el calendario y la agenda del centro
Ver todas las citas y reservas
Crear citas manualmente (presencial o telefónica)
Modificar citas existentes
Cambiar profesionales asignados
Cambiar salas o recursos
Reprogramar citas
Cancelar citas
Marcar citas como realizadas o no asistidas
Gestionar clientes
Ver pagos realizados
Registrar pagos realizados en el centro
Gestionar devoluciones
Gestionar disponibilidad de profesionales
Gestionar disponibilidad de salas o recursos
Bloquear horarios (vacaciones, mantenimiento, etc.)

El panel de administración tiene prioridad sobre el sistema automático de reservas, por lo que cualquier cita podrá ser modificada manualmente por el personal del centro.

4. Objetivo del proyecto

El objetivo del proyecto es crear una aplicación web completa que incluya:

Web corporativa profesional
Sistema de reservas online con pago de señal
Motor de asignación automática de servicios y profesionales
CRM interno para la gestión del negocio
Calendario inteligente sincronizado con las reservas online
Gestión de clientes, citas, pagos y recursos

Este proyecto servirá como:

Proyecto de portfolio profesional
Proyecto de aprendizaje de desarrollo web
Base para un sistema real que podría utilizar un centro de estética
5. Alcance inicial (versión 1)

La primera versión del sistema incluirá:

Selección de servicios online
Cálculo automático de duración total
Búsqueda de huecos disponibles
Asignación automática de profesionales y recursos
Pago de señal (20%) para confirmar la reserva
Confirmación por email
Panel de administración con:
Calendario
Gestión de citas
Gestión de clientes
Visualización de pagos
Modificación manual de citas

Funciones más avanzadas como cuentas de usuario, cambios de cita desde la web o estadísticas se dejarán para versiones futuras.

1. Introducción

Este documento define el catálogo de servicios de Glow Up Studio que será utilizado por:

La web pública
El sistema de reservas online
El CRM interno
El motor de asignación automática de profesionales y recursos

Cada servicio tiene definidos:

Categoría
Precio
Duración estimada
Margen de tiempo
Profesionales que pueden realizarlo
Recurso o sala necesaria

La duración definida en cada servicio será la duración utilizada por el sistema para bloquear agenda.

Se establece un margen estándar de 5 minutos por servicio para preparación del material, limpieza y organización del puesto.

4. Reglas del sistema relacionadas con servicios
Cada servicio tiene una duración total que será la que el sistema utilice para bloquear agenda.
El margen de 5 minutos ya está incluido en la duración total del sistema.
Los servicios pueden combinarse entre sí en una misma reserva.
El sistema puede cambiar el orden de los servicios para optimizar la agenda.
El sistema debe comprobar siempre:
disponibilidad de profesional
disponibilidad de recurso o cabina
Se permite una espera máxima de 15 minutos entre servicios combinados.
Si la espera supera 15 minutos, esa combinación no se mostrará como disponible.
El sistema debe intentar siempre minimizar las esperas entre servicios.
